## 🌌 Overview

This portfolio isn't just a resume; it's a digital dossier designed to "WOW" at first glance. It utilizes cutting-edge web technologies to create a cinematic user experience, featuring dynamic 3D elements, smooth micro-animations, and a highly structured information hierarchy.

### Core Philosophy
- **Impact-First Storytelling**: Every project and experience is framed by its outcome (e.g., "20% Default Reduction").
- **Visual Excellence**: A strictly curated color palette using deep blacks (Obsidian) and vibrant golds to convey luxury and authority.
- **Technical Depth**: Transparent tech stacks for every role and project, showing exactly *how* results were achieved.

## ✨ Key Features

### 1. Cinematic 3D Hero Scene
Powered by **Three.js**, the hero section features a reactive 3D environment that sets a professional, high-tech tone immediately upon entry.

### 2. Interactive Journey Timeline
A premium S-curve timeline that weaves between experience cards, tracking the evolution from Data Scientist to Technical Product Manager.

### 3. Credentials Dossier
A high-density grid showcasing professional certifications from **Microsoft**, **Databricks**, and **Oracle**, with interactive hover states revealing brand-specific identities.

### 4. Experiments & Project Gallery
A categorized showcase of AI, Fintech, and Data projects, each featuring a "Problem-Approach-Impact" breakdown.

### 5. Intellectual Property Showcase
Formalized sections for **Patents** (AI Smart Glasses) and **IEEE Publications**, highlighting a commitment to research and innovation.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/) (Functional Components, Hooks)
- **Build Tool**: [Vite 8](https://vitejs.dev/) (Ultra-fast HMR)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/) (Modern utility-first CSS)
- **Animations**: [Framer Motion](https://www.framer.com/motion/) (Spring physics, layout transitions)
- **3D Graphics**: [Three.js](https://threejs.org/) (Hero scene orchestration)
- **Icons**: [Lucide React](https://lucide.dev/) (Consistent, minimal iconography)
- **Routing**: [React Router 7](https://reactrouter.com/) (Client-side navigation)
- **SEO**: [React Helmet Async](https://github.com/staylor/react-helmet-async) (Dynamic meta tags)

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/CHAITANYA-2002/portfolio.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```
   Open `http://localhost:5173` in your browser.

### Available Scripts
- `npm run dev`: Starts the Vite development server.
- `npm run build`: Compiles the application for production.
- `npm run preview`: Locally previews the production build.
- `npm run lint`: Runs ESLint for code quality checks.
