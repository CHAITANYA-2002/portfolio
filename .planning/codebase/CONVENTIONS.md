# Conventions

## Code Style
- **Functional Components**: React functional components with hooks.
- **Lazy Loading**: `Suspense` and `lazy()` for section-level separation.
- **Naming**: PascalCase for components, camelCase for variables/functions.

## Styling
- **Tailwind CSS 4**: Preference for utility classes over plain CSS.
- **Theme Variables**: CSS variables in `globals.css` for theme colors and fonts.
- **Glassmorphism**: `.bespoke-card` class for a premium translucent effect.

## Animation
- **Framer Motion**: Standardized entrance animations (opacity, y-offset).
- **Smoothness**: Emphasis on 0.7s to 1s durations for a premium feel.
