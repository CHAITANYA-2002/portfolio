# Structure

## Directory Tree
- `.planning/`: Project tracking and mapping.
- `src/`: Source code.
  - `assets/`: Static assets (images, logos).
  - `components/`: UI components and sections.
  - `data/`: `portfolio.js` (Central data storage).
  - `styles/`: `globals.css` (Tailwind 4 + Custom styles).
  - `App.jsx`: Main entry point and layout.
  - `main.jsx`: React hydration.
- `public/`: Public assets.
- `tailwind.config.js`: Tailwind configuration (v4 also uses globals.css for theme).
